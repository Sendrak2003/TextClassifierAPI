Примеры команд для тестирования:
--------------------------------

curl:
curl -X POST http://localhost:3000/classify -H "Content-Type: application/json" -d "{\"text\": \"Закажи пиццу из Domino's на завтра в 18:00, ZIP 90210\"}"

PowerShell:
Invoke-WebRequest -Uri "http://localhost:3000/classify" -Method POST -ContentType "application/json" -Body '{"text": "Закажи пиццу из Domino'\''s на завтра в 18:00, ZIP 90210"}'

JavaScript:
fetch('http://localhost:3000/classify', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ text: 'Закажи пиццу из Domino\'s на завтра в 18:00, ZIP 90210' })
})

Примечания:
-----------
- Все ZIP коды являются примерными
- Время может быть указано в различных форматах
- Бренды могут быть написаны по-разному
- Категории товаров могут варьироваться
- API поддерживает русский и английский языки
